Áreas de Interesse

Tenho muito interesse em Inteligência Artificial, Deep Learning, Machine Learning, Data Science, Banco de Dados, Análise de Dados e Back-end. De modo geral, gosto de trabalhar com dados e extrair algo útil dos mesmos.
Estou desenvolvendo uma Iniciação Científica junto do meu orientador, onde, como produto final, é esperado um modelo capaz de realizar a classificação de bovinos infectados com os parasitas "Mosca-dos-chifres". Para isso, estou estudando métodos conhecidos como CNN e SVM. Assim que obtiver algum fruto deste projeto, irei postar aqui. Minha intenção é, após o desenvolvimento do mesmo, disponibilizar uma API pública para o consumo da mesma, para realizar testes de classificações, e também um modelo gráfico, onde ficará disponivel na seção de Projetos.
